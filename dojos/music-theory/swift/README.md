# Music Theory in Swift!

This repo contains a representation of basic music theory scales in Swift

# Running

## Requirements
You'll need to have `swiftc` installed

## Compiling
`swiftc -o OUTPUT Main.swift`

## Executing
### Testing
`./OUTPUT`

### With parameters
```sh
./OUTPUT C

> Minor: C D Eb F G Ab Bb 
Major: C D E F G A B 

```
