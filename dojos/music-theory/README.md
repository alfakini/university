# Music Theory

## Problem

Given the musical notes (C, C#, D, D#, E, F, F#, G, G#, A, A#, B)
we want to build the Major and Minor scales.

## Requirements

### Input

Receives the root note

### Output

Returns the Major and Minor scale

### Tests

Must test for:

- C Major scale
- C Minor scale

### Example

| Input | Output                  |
| :---: | :---                    |
| C     | Major: C D E F G A B    |
|       | Minor: C D Eb F G Ab Bb |
